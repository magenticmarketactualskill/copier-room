# Copier-Room Architecture Project

This repository contains research and documentation for the Copier-Room architecture, a hexagonal architecture implementation that uses JSON-RPC-LD for framework-agnostic communication.

## Architecture Overview

The Copier-Room architecture is a hexagonal architecture implementation that disintegrates functionality typically integrated within web frameworks, providing a framework-agnostic service that communicates via JSON-RPC-LD.
